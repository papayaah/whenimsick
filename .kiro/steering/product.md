# Product Overview

When I'm Sick is a privacy-first health tracking web application that helps users track symptoms and receive AI-powered health insights completely offline.

## Core Features

- **Symptom Tracking**: Select from common/uncommon symptoms with severity levels
- **AI Analysis**: Instant analysis using Chrome's built-in AI (Gemini Nano) or Gemini API - no data sent to external servers
- **Episode Tracking**: Multi-day illness progression with automatic episode detection and trend analysis
- **Medical Glossary**: Interactive tooltips for medical terms with definitions
- **Privacy-First**: All data stored locally in browser LocalStorage

## Key Principles

- **Educational Only**: Not medical advice - for educational purposes only
- **Privacy**: 100% private - no data leaves the user's device (when using Chrome AI)
- **Offline-First**: Works completely offline with Chrome's built-in AI
- **User Control**: Users own their health data stored locally

## Target Users

Individuals who want to track their health symptoms privately and receive educational health insights without sharing data with external services.
